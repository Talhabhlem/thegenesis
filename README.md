## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content in `index.html`
4. Modify colors and styling in `styles.css`
5. Add your own functionality in `script.js`

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more colors ... */
}
```

### Services
Edit the service cards in the "Services Section" of `index.html`

### Pricing
Modify the pricing cards in the "Pricing Section" of `index.html`

### Contact Information
Update contact details in the "Contact Section" and "Footer"
